# NovaTech E-Commerce Store

A modern e-commerce web application built with HTML, CSS, and JavaScript.

## Features

* Product catalog
* Product search
* Category filtering
* Shopping cart
* Wishlist system
* Order checkout flow
* Customer reviews section
* Responsive design
* LocalStorage persistence

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
